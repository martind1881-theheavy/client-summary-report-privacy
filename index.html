🔒 Privacy Policy

Client Summary Assistant  ·  Chrome Extension by Innov8Edge

Last Updated: March 2026

Simple Summary: Client Summary Assistant stores all your data locally on your device using Chrome's built-in storage. Your OpenAI API key is stored only on your device and is sent exclusively to OpenAI when generating summaries. We never sell your data, never collect it on our own servers, and you can delete everything at any time.

1. Information We Collect

Client Summary Assistant collects and stores the following information locally on your device:

  • Summary History: Client names, summary types, generated text, and dates (capped at 50 records; oldest removed automatically)
  • API Key: Your OpenAI API key, stored locally in Chrome's encrypted storage
  • Usage Statistics: Token counts, time savings estimates, and summary counts — stored locally for your Dashboard
  • Feedback: Star ratings, improvement suggestions, and optionally your email address if provided voluntarily
  • Preferences: Time estimate settings, Creator Mode state, and onboarding completion flags

We do not collect your name, billing information, browsing history, or any data beyond what is listed above.

2. How We Use Your Information

  • Summary Generation: Form field values are assembled into a prompt and sent to OpenAI to generate your summary
  • Dashboard: Usage statistics, history, and feedback are displayed back to you within the extension
  • Time Savings: Locally calculated estimates based on your configured time-per-summary settings
  • Walkthrough Tour: A flag is stored locally to remember whether you have completed the guided tour
  • Feedback Follow-Up: If you voluntarily provide your email with feedback, Innov8Edge may use it to respond to your suggestion

3. Data Storage

Local Storage Only

All data is stored on your device using chrome.storage.local. This data does not leave your device except in the two cases described in Section 4.

No Cloud Sync

Client Summary Assistant does not operate its own servers and does not back up your data to any cloud service. Your summary history, statistics, and preferences exist only on the device where the extension is installed.

Data Retention

Summary history is automatically capped at 50 records. When the limit is reached the oldest entry is silently removed. You can delete all stored data at any time via Options → Data Management.

4. Third-Party Services

OpenAI

When you generate a summary, the content of your form fields is sent to OpenAI's API using your personal API key. This is the only time any data leaves your device.

  • Your API key is stored locally and sent directly to OpenAI — Innov8Edge never sees or stores it
  • OpenAI Privacy Policy: openai.com/policies/privacy-policy
  • OpenAI Terms of Use: openai.com/policies/terms-of-use

BrowserBill

The extension uses BrowserBill to manage subscription status. BrowserBill may process your browser identity to verify your subscription. View their privacy policy at browserbill.com.

This extension uses OpenAI's GPT-4 to generate client summaries. Please be aware:
  • AI responses may contain inaccuracies or omissions
  • Generated summaries should be reviewed before use in a professional context
  • AI output does not constitute legal, financial, or compliance advice
  • Innov8Edge is not responsible for decisions made based on AI-generated content

6. Data Sharing

We do not sell, rent, or share your personal data with third parties.

Data is only transmitted to:

  • OpenAI's API — when you actively click Generate Summary
  • BrowserBill — to verify your subscription status

No analytics platforms, advertising networks, or data brokers receive any information from this extension.

7. Your Rights

  • View: All stored data is visible within the Dashboard and History tabs
  • Edit: Update preferences and settings at any time via the Options page
  • Delete: Use Options → Data Management → Clear All Data to permanently wipe everything
  • API Key: Remove your stored API key at any time using the Clear Key button in the popup

8. Data Security

  • HTTPS encryption for all communications with OpenAI and BrowserBill
  • Local-first architecture — your data stays on your device by default
  • API keys stored in Chrome's encrypted chrome.storage.local
  • No server-side processing, logging, or analytics by Innov8Edge

9. Children's Privacy

Client Summary Assistant is intended for professional use by adults. It is not directed at children under the age of 13. We do not knowingly collect data from children.

10. Changes to This Policy

We may update this Privacy Policy to reflect changes in functionality or legal requirements. The "Last Updated" date at the top of this page will be revised accordingly. Continued use of the extension after changes are posted constitutes acceptance of the updated policy.

11. Permissions Explained

Client Summary Assistant requests the following Chrome permissions:

  • storage Save your API key, summary history, statistics, and preferences locally
  • tabs Open the Dashboard, Welcome screen, and result pages in new tabs
  • activeTab Read the URL of the current tab for context when generating summaries
  • scripting Inject content into the active tab when needed for summary context
  • identity Support Google sign-in for subscription verification via BrowserBill
  • host permissions Communicate with OpenAI's API and BrowserBill's servers

We request only the minimum permissions necessary for core functionality.
